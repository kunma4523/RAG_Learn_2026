# Data Download Scripts
# =====================

This directory contains scripts to download sample datasets.

## Available Scripts

### download_nq.py
Download Natural Questions dataset.

```bash
python data/scripts/download_nq.py
```

### download_triviaqa.py
Download TriviaQA dataset.

```bash
python data/scripts/download_triviaqa.py
```

### download_hotpotqa.py
Download HotpotQA dataset.

```bash
python data/scripts/download_hotpotqa.py
```
