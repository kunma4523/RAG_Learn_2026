# RAG Classic Papers
# =================

## Must-Read Papers

### 1. Retrieval-Augmented Generation (RAG)

**Title**: Retrieval-Augmented Generation for Knowledge-Intensive NLP Tasks
**Authors**: Patrick Lewis, Ethan Perez, Aleksandra Piktus, Fabio Petroni, Vladimir Karpukhin, Naman Goyal, etc.
**Year**: 2020
**arXiv**: [2005.11401](https://arxiv.org/abs/2005.11401)
**Venue**: NeurIPS 2020
**Abstract**: We explore a general-purpose retrieval-augmented generation (RAG) model.

### 2. Dense Passage Retrieval (DPR)

**Title**: Dense Passage Retrieval for Open-Domain Question Answering
**Authors**: Tikt, Omer Levy, Sebastian Goodman, etc.
**Year**: 2020
**arXiv**: [2004.09541](https://arxiv.org/abs/2004.09541)
**Venue**: EMNLP 2020
**Abstract**: We show that dense retrievers outperform BM25.

### 3. REALM

**Title**: REALM: Retrieval-Augmented Language Model Pre-Training
**Authors**: Kelvin Guu, Kenton Lee, Zora Tung, etc.
**Year**: 2020
**Venue**: ICML 2020

### 4. Contriever

**Title**: Contriever: Continuous Contrastive Retriever
**Authors**: Gabriel L. S. F. Pereira, etc.
**Year**: 2022
**arXiv**: [2206.14258](https://arxiv.org/abs/2206.14258)

### 5. In-Context RALM
