# Notebooks
# =========

This directory contains Jupyter notebooks for learning RAG.

## Notebook Categories

### Getting Started
- `1_basic_rag.ipynb` - Your first RAG application
- `2_langchain_intro.ipynb` - LangChain basics
- `3_llamaindex_intro.ipynb` - LlamaIndex basics

### Retrieval
- `4_bm25_retrieval.ipynb` - BM25 retrieval
- `5_dense_retrieval.ipynb` - Dense embedding retrieval
- `6_hybrid_retrieval.ipynb` - Combining sparse and dense

### Architectures
- `7_standard_rag.ipynb` - Standard RAG pipeline
- `8_conversational_rag.ipynb` - Conversational RAG
- `9_hyde.ipynb` - HyDE implementation
- `10_self_rag.ipynb` - Self-RAG
- `11_agentic_rag.ipynb` - Agentic RAG
- `12_graph_rag.ipynb` - GraphRAG

### Advanced Topics
- `13_reranking.ipynb` - Cross-encoder reranking
- `14_query_rewriting.ipynb` - Query rewriting
- `15_evaluation.ipynb` - RAG evaluation

### Projects
- `project_1_qa_system.ipynb` - Build a QA system
- `project_2_chatbot.ipynb` - Build a chatbot
- `project_3_agent.ipynb` - Build an agent

## How to Use

1. Install dependencies: `pip install -r requirements.txt`
2. Start Jupyter: `jupyter notebook`
3. Open a notebook and follow along

## Requirements

- Python 3.10+
- Jupyter Lab or Jupyter Notebook
- See requirements.txt for package dependencies
