# Python环境搭建

本节介绍如何搭建Python开发环境。

## 1. 安装Python

### 方式一：从官网安装

访问 [Python官网](https://www.python.org/) 下载最新版本（建议3.10或更高）。

### 方式二：使用包管理器

```bash
# macOS
brew install python3

# Ubuntu/Debian
sudo apt update
sudo apt install python3 python3-pip

# Windows
# 从 Microsoft Store 安装 Python 3.10+
```

## 2. 版本管理

推荐使用 pyenv 管理多个Python版本：

```bash
# macOS/Linux 安装 pyenv
curl https://pyenv.run | bash

# 安装指定版本
pyenv install 3.10.13

# 设置全局版本
pyenv global 3.10.13
```

## 3. 虚拟环境

### 使用 venv（推荐）

```bash
# 创建虚拟环境
python -m venv myenv

# 激活环境
source myenv/bin/activate  # Linux/Mac
myenv\Scripts\activate     # Windows

# 退出环境
deactivate
```

### 使用 uv（高性能）

```bash
# 安装 uv
pip install uv

# 创建并激活环境
uv venv
source .venv/bin/activate
```

### 使用 conda

```bash
# 创建环境
conda create -n rag-learn python=3.10

# 激活环境
conda activate rag-learn
```

## 4. 常用命令

```bash
# 查看Python版本
python --version

# 查看pip版本
pip --version

# 升级pip
python -m pip install --upgrade pip

# 安装包
pip install package_name

# 导出依赖
pip freeze > requirements.txt

# 从文件安装
pip install -r requirements.txt
```

## 5. IDE推荐

- **VS Code** - 免费、功能强大
- **PyCharm** - 专业Python IDE
- **Jupyter Notebook** - 数据科学/机器学习

### VS Code 扩展推荐

```bash
# 安装 Python 扩展
code --install-extension ms-python.python

# 安装 Jupyter 扩展
code --install-extension ms-toolsai.jupyter
```
