## 🚀 快速开始

建议按顺序学习本模块，确保掌握以下核心概念：

1. **Python与深度学习基础**
   - PyTorch张量操作
   - Transformers库使用

2. **检索基础**
   - 倒排索引原理
   - 稀疏与密集检索区别

3. **向量表示**
   - 词向量与句子嵌入
   - 相似度计算方法
