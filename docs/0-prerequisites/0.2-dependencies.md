# 依赖安装

本节介绍如何安装RAG项目所需的依赖。

## 1. 核心依赖

```bash
# 创建虚拟环境后，安装核心依赖
pip install torch>=2.0.0
pip install transformers>=4.35.0
pip install sentence-transformers>=2.2.0
```

## 2. RAG框架

```bash
# LangChain 生态
pip install langchain>=0.1.0
pip install langchain-community>=0.0.10
pip install langchain-core>=0.1.10

# LlamaIndex
pip install llama-index>=0.9.0
pip install llama-index-core>=0.9.0
```

## 3. 向量数据库

```bash
# Faiss
pip install faiss-cpu>=1.7.4
pip install faiss-gpu>=1.7.4

# 其他向量库
pip install pinecone-client>=3.0.0
pip install weaviate-client>=4.0.0
pip install qdrant-client>=1.7.0
pip install chromadb>=0.4.0
```

## 4. LLM客户端

```bash
pip install openai>=1.0.0
pip install anthropic>=0.18.0
pip install google-genai>=0.3.0
pip install cohere>=5.0.0
pip install tiktoken>=0.5.0
```

## 5. 文档处理

```bash
pip install pypdf>=3.17.0
pip install python-docx>=1.1.0
pip install python-pptx>=0.6.23
pip install unstructured>=0.10.0
pip install langchain-text-splitters>=0.0.1
```

## 6. 评估工具

```bash
pip install ragas>=0.1.0
pip install deepeval>=0.1.0
pip install bert-score>=0.3.13
pip install rouge-score>=0.1.2
```

## 7. 一键安装

```bash
# 从项目根目录
pip install -r requirements.txt
```

## 8. 验证安装

```python
import torch
import transformers
import sentence_transformers
import langchain

print(f"PyTorch: {torch.__version__}")
print(f"Transformers: {transformers.__version__}")
print(f"LangChain: {langchain.__version__}")
```
